Scratch

This a visual programming language targeted primarly for enhancing children of school age programming skills and one of the friendliest programming languages ever design for children.

Python

Python is an interpreted, object-oriented, high-level programming language with dynamic semantics, Python's simple, easy to learn syntax emphasizes readability which supports modules and packages

App Inventor

App Inventor is a visual, blocks language for building Android Apps, a product of the University of San Francisco's Democratize Computing Lab and is funded by the NSF and WM Keck foundation.







Tynker

is a complete learning system that teaches kids to code. Kids begin experimenting with visual blocks, then progress to JavaScript, Swift, and Python as they design games, build apps, and make incredible projects.

HTML & CSS

HyperText Markup Language, gives content structure and meaning by defining that content as, for example, headings, paragraphs, or images. CSS is know as Cascading Style Sheets, is a presentation language created to style the appearance of content.

Photoshop

is Adobe's photo editing, image creation and graphic design software. The software provides many image editing features for raster (pixel-based) images as well as vector graphics. It uses a layer-based editing system that enables image creation and altering with multiple overlays that support transparency.
