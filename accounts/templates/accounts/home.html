Welcome to GWD Code Club

GWD Code Club is a network of coding clubs for children of ages 8-18.


Get Involved!

We live in a digital age, therefore the knowledge of some of these basic IT skills such as computer programming (coding), app development and web development makes one stay relevant in the 21st century. The demand for these and other technical skills will only grow bigger. So it becomes imperative to expose pupils and students to basic IT skills early enough to give them an edge in this highly competitive computer age.

Why Code Club?


Educationschool

It’s no secret that coding helps kids develop skills they need to be successful in school, e.g “critical thinking” It’s a concept that includes several key steps—which is great because, just like any skill set, critical thinking can be developed with practice. As they code, Kids learn what happens when a given set of blocks is combined or altered.

innovationmemory

Like critical thinking, creativity brings innovation, that is a skill that can be developed! One of the key steps in the creative process is experimentation. With Tynker or Scratch, kids can test out their imaginative ideas!.

Oollaborationcollections

While it’s important to have “alone time” while working on a coding project, much can be gained—and given—when kids team up with others, they demonstrate resilience and improve their coding skills while learning how to collaborate with others

communicationcomment

Dynamic communication that uses a variety of media is becoming increasingly common in today’s society. Kids need to learn how to stand up in front of other people and articulate their ideas and how to use digital tools to share information and there tell stories.
